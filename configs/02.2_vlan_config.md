#  Configuration Phase 2.2 : Routage Zone Opérationnelle (Dépôt)

Ce document détaille la mise en œuvre du routage **Router-on-a-Stick** pour la zone "Depot" (Côté Gauche) ainsi que la configuration du switch d'accès.

---

##  1. Configuration du Routeur Central (R1)
*Mise en place des sous-interfaces pour assurer le routage inter-VLAN du site distant.*

```
interface GigabitEthernet 0/0/1
 description LINK_TO_SWITCH_DEPOT
 no shutdown                        ! L'interface physique doit être UP

 --- Sous-interface VLAN 70 (Partners) ---
interface GigabitEthernet 0/0/1.70
 description GATEWAY_PARTNERS
 encapsulation dot1Q 70             ! Tagging 802.1Q
 ip address 172.16.70.1 255.255.255.0

 --- Sous-interface VLAN 80 (Logistics) ---
interface GigabitEthernet 0/0/1.80
 description GATEWAY_LOGISTICS
 encapsulation dot1Q 80
 ip address 172.16.80.1 255.255.255.0

 --- Sous-interface VLAN Natif ---
interface GigabitEthernet 0/0/1.99
 description NATIVE_VLAN
 encapsulation dot1Q 99 native      ! Doit correspondre au Native VLAN du switch
 
 --- Route de retour vers le Siège ---
! Permet au routeur de renvoyer le trafic vers le Switch L3 (Siège)
ip route 192.168.0.0 255.255.0.0 10.0.0.2
```
