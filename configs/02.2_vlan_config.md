#  Configuration Phase 2.2 : Segmentation & Routage (Zone Dépot)

Ce document détaille la mise en œuvre du routage **Router-on-a-Stick** pour la zone "Dépot" (Côté Gauche) ainsi que la configuration du switch d'accès.


##  1. Configuration du Routeur Central (R1)
#### Mise en place des sous-interfaces pour assurer le routage inter-VLAN du site distant.

```
interface GigabitEthernet 0/0/1
 description LINK_TO_DEPOT
 no shutdown                        ! L'interface physique doit être UP

 --- Sous-interface VLAN 70 (Partners) ---
interface GigabitEthernet 0/0/1.70
 description GATEWAY_PARTNERS
 encapsulation dot1Q 70             ! Tagging 802.1Q
 ip address 172.16.70.1 255.255.255.0

 --- Sous-interface VLAN 80 (Logistics) ---
interface GigabitEthernet 0/0/1.80
 description GATEWAY_LOGISTICS
 encapsulation dot1Q 80
 ip address 172.16.80.1 255.255.255.0

 --- Sous-interface VLAN Natif ---
interface GigabitEthernet 0/0/1.99
 description NATIVE_VLAN
 encapsulation dot1Q 99 native      ! Doit correspondre au Native VLAN du switch
 
 --- Route de retour vers le Siège ---
! Permet au routeur de renvoyer le trafic vers le Switch L3 (Siège)
ip route 192.168.0.0 255.255.0.0 10.0.0.2
```
##  2. Configuration du Switch Accès
### Préparation des VLANs et du lien Trunk vers le routeur.
#### Création des VLANs locaux
```
vlan 70
 name PARTNERS
vlan 80
 name LOGISTICS
vlan 99
 name NATIVE_VLAN
vlan 999
 name UNUSED_PORTS
```
#### Configuration du Trunk et des Accès
```
 --- Uplink vers le Routeur ---
interface GigabitEthernet 0/1
 description LINK_ROUTER
 switchport mode trunk
 switchport trunk native vlan 99
 switchport trunk allowed vlan 70,80,99

 --- Ports Utilisateurs (VLAN 70) ---
interface range FastEthernet 0/1 - 5
 description ACCESS_PARTNERS
 switchport mode access
 switchport access vlan 70
 spanning-tree portfast
 spanning-tree bpduguard enable

--- Ports Utilisateurs (VLAN 80) ---
interface range FastEthernet 0/6 - 10
 description ACCESS_LOGISTICS
 switchport mode access
 switchport access vlan 80
 spanning-tree portfast
 spanning-tree bpduguard enable
```
##  3. Configuration du Switch Accès
#### Désactivation globale des ports non utilisés
```
interface range FastEthernet 0/11 - 24, GigabitEthernet 0/2
 description UNUSED_DISABLED
 switchport mode access
 switchport access vlan 999
 shutdown
```
