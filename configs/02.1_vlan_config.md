#  Configuration Phase 2.1 : Segmentation & Routage (Zone Siège)

Ce document détaille la configuration du **Switch Cœur (S3-L3)** et des **Switchs d'accès (S1/S2)** pour la mise en place du réseau local du Siège (Côté droit).


##  1. Infrastructure VLAN
#### Création des domaines de diffusion et isolation des services.

```
 --- Création de la base de données VLAN ---
vlan 10
 name ADMIN
vlan 20
 name PROD
vlan 30
 name SALES
vlan 40
 name GUEST
vlan 99
 name NATIVE_VLAN           ! Gestion des Trunks
vlan 999
 name UNUSED_PORTS          ! Sécurité (VLAN Blackhole)
```
##  2. Routage Inter-VLAN (S3-L3)
#### Activation du routage IP et configuration des interfaces virtuelles (Gateway).
```
ip routing                  ! Activation du routage

interface Vlan 10
 description GW_ADMIN
 ip address 192.168.10.1 255.255.255.0
 no shutdown

interface Vlan 20
 description GW_PROD
 ip address 192.168.20.1 255.255.255.0
 no shutdown

interface Vlan 30
 description GW_SALES
 ip address 192.168.30.1 255.255.255.0
 no shutdown

interface Vlan 40
 description GW_GUEST
 ip address 192.168.40.1 255.255.255.0
 no shutdown
```
##  3. Routage Inter-VLAN (S3-L3)
#### Configuration du port routé et de la route par défaut vers le WAN.
```
 --- Port Routé (Liaison L3) ---
interface GigabitEthernet 0/1
 description LINK_TO_ROUTER
 no switchport                ! Interface configurée en couche 3
 ip address 10.0.0.2 255.255.255.252
 no shutdown

 --- Routage Statique ---
Redirection du trafic vers le routeur R1 (next hop)
 ip route 0.0.0.0 0.0.0.0 10.0.0.1
```
## 4. Trunks et Accès (S1/S2)
#### Transport des VLANs et sécurisation des ports terminaux.

Liaisons Trunks 
```
interface range FastEthernet 0/21 - 24
 switchport trunk encapsulation dot1q
 switchport mode trunk
 switchport trunk native vlan 99
 switchport trunk allowed vlan 10,20,30,40,99
```
#### Configuration des ports d'accès
```
 --- Exemple pour les postes Admin ---
interface range FastEthernet 0/1 - 5
 description ACCESS_VLAN_10
 switchport mode access
 switchport access vlan 10
 spanning-tree portfast        ! Accélération de la convergence
 spanning-tree bpduguard enable ! Blocage des BPDUs indésirables
```
## 5. Sécurisation des ports inutilisés 
#### Application de la politique de sécurité pour les ports non assignés.
```
--- Exemple sur S1 ---
interface range FastEthernet 0/11 - 20, GigabitEthernet 0/1 - 2
 description UNUSED
 switchport access vlan 999
 shutdown                      ! Port désactivé administrativement
```
